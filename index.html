iOS-style Placeholder jQuery Plugin

Inspired by Robert Nyman in his alternative approach to the HTML5 placeholder behaviour, and Recurly's nice form examples, this plugin aims to mimic the behaviour of the placeholder behaviour on iOS form fields i.e. keep the placeholder text on focus, and only remove upon entering text.

Apparently the iOS behaviour is now used in OS X Lion, so if you're using Lion, you probably won't see much difference in the demos below.
